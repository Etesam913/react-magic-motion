# `<MagicTabSelect />`

### Overview

- This component is used for animating the tab selection indicator.

An example of this component is seen on the [Tabs example](/applications/tabs).

### Props

```ts
interface MagicTabSelectProps {
  id: string;
  children: JSX.Element;
  transition?: Transition;
}
```

| Prop name                                                 | Default value        | Description                                   |
| :-------------------------------------------------------- | :------------------- | :-------------------------------------------- |
| `id`                                                      | undefined            | Used to identify the component being animated |
| [`transition`](https://www.framer.com/motion/transition/) | set by framer-motion | The transition animation for the tab select.  |
